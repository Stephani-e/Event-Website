# Event Website

It showcases key sections an event brand would need, from services and gallery to contact and booking.

## Features

- Multiple pages for different company needs (home, services, about, contact, etc.).
- Reusable layout template shared across all pages.
- Fully static: no frameworks, just semantic HTML and modern CSS.
- Responsive structure designed to work across common screen sizes.

## Tech Stack

- HTML5
- CSS3 (custom styling, no libraries)
- Deployed on Vercel
